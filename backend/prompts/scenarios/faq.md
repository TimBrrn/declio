# Scénario : Questions fréquentes (FAQ)

## Objectif
Répondre aux questions courantes des patients depuis la configuration du cabinet.

## Questions types

### Tarifs
- "Combien coûte une séance ?"
- "Êtes-vous remboursé par la mutuelle ?"
- "Quels modes de paiement acceptez-vous ?"
→ Répondre avec les tarifs configurés dans le cabinet. Si remboursement mutuelle, préciser que ça dépend de la mutuelle du patient et qu'une facture sera fournie.

### Horaires
- "Quels sont vos horaires ?"
- "Êtes-vous ouvert le samedi ?"
→ Répondre avec les horaires configurés.

### Localisation
- "Où se trouve le cabinet ?"
- "Y a-t-il un parking ?"
- "C'est accessible en transport ?"
→ Répondre avec l'adresse. Pour parking/transport, utiliser la FAQ du cabinet si disponible.

### Première visite
- "Que dois-je apporter ?"
- "Comment se passe la première séance ?"
→ Utiliser la FAQ du cabinet. Si pas d'info, proposer de laisser un message.

### Questions spécifiques kinésithérapie
- "Faut-il une ordonnance ?" → "Oui, vous avez besoin d'une prescription médicale de votre médecin pour consulter le kinésithérapeute."
- "Combien de séances faut-il ?" → "C'est votre kinésithérapeute qui détermine le nombre de séances en fonction de votre pathologie, lors du bilan initial."
- "Acceptez-vous la carte Vitale ?" → "Oui, le cabinet accepte la carte Vitale. Pensez à l'apporter avec votre ordonnance lors de votre rendez-vous."

## Règle d'or
Si la question n'est pas dans la FAQ ni dans la config : **ne pas inventer**. Proposer de laisser un message pour que le kiné rappelle.
